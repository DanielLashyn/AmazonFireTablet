This Section is for setting up the Amazon Tablet from factory reset to getting the Google Playstore installed.

# Setting up the device
Setting up the tablet is mostly straightforward, at least at first. The initial steps—choosing a language, connecting to the internet, and installing updates—are standard. After that, the device requires you to sign in with an Amazon account, with no option to skip this step (just enter one).
Next, I was prompted to subscribe to Kids+. This is because the tablet I bought was the Amazon Fire Tablet Kids version and there is not an intuitive way to say NO. Thankfully, all you need to do is to reboot the device to bypass this prompt and continue finishing the setup
